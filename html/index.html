Arquitectura de la Práctica

Este nodo Apache funciona como uno de los dos backends configurados. Las peticiones son gestionadas por un proxy inverso Nginx que aplica un algoritmo de balanceo Round Robin.

El contenido que estás viendo se sirve desde un volumen compartido, asegurando que ambos nodos web muestren la misma información sincronizada.
